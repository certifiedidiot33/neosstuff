texture packs allow users to create their own textures to use ingame.

How to create a texture pack

firstly you have to create the textures e.g making them in photoshop
secondly you have to upload each texture front,back,left,right,top,bottom of each block
then download the module script that has the values (note, if you leave a texture blank it will be its default texture)
